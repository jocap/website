Instapaper bookmarklet without JavaScript

(October, 2018)

For the most part, I browse the web with JavaScript disabled by default (via NoScript). I also use Instapaper to save articles for later. Unfortunately, the bookmarklet1 provided by Instapaper requires JavaScript to be enabled on the site to be saved.

For this reason, I cobbled together my own bookmarklet, inspired by the ones provided by Pinboard (an even better service):

javascript:void(open('https://www.instapaper.com/hello2?u='+encodeURIComponent(location.href)+'&t='+(document.title),'Instapaper','toolbar=no,width=500,height=350'))

Here it is as a link that you can drag to your bookmarks: Add to Instapaper

  1. “A bookmarklet is a bookmark stored in a web browser that contains JavaScript commands that add new features to the browser.” – Wikipedia ↩
